## ğŸ› ï¸ How to Use the Cancel Button

1. If a bill has been created but should not be processed, **click the Cancel button**.
2. The system will prompt you to **enter a reason** for canceling.
3. Enter the appropriate reason and **click the Proceed button**.
4. A confirmation message will appear:
   > "Are you sure you want to cancel the bill?"
5. To finalize:
   - Click **Confirm** to cancel the bill.
   - Click **Cancel** if you decide not to cancel.
