# Without Bill Refund

This feature lets you process **refunds without linking** to a previous bill.

## 📝 When to Use

- Customer returns items without an invoice.
- Original bill is lost or not traceable.

---

## 🛠️ How to Use

1. Click the **Refund** button.
2. Select **Refund Type**: _Without Bill Refund_.
3. Scan or search the item(s) being returned.
4. Enter correct **quantity** and **price**.
5. Review refund details.
6. Select a **payment method** to process refund.
