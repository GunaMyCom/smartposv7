# 💱 Foreign Exchange

The **Foreign Exchange** feature allows customers to pay their bill using **foreign currencies**. The application automatically converts the selected currency to the local currency and processes the payment.

## ✅ What You Can Do

1. Accept payments in **multiple foreign currencies**.  
2. Automatically **calculate exchange rates** and balance due.  
3. Accept **full or partial** payment in a selected foreign currency.  
4. Support for currencies like:
   - 🇦🇪 UAE Dirhams (AED)
   - 🇮🇳 Indian Rupees (INR)
   - 🇺🇸 US Dollars (USD)
   - 🇪🇺 Euros (EUR)
   - 🇰🇼 Kuwaiti Dinar (KWD)
   - 🇴🇲 Omani Riyal (OMR)

---

## 🛠️ How to Use

1. Create a new bill by adding the desired items.  
2. Click the **Foreign Exchange** button on the billing screen.  
3. A list of available foreign currencies will appear (e.g., AED, INR, USD).  
4. Select the currency the customer wants to use.  

---

### 💵 Entering Payment

5. A new screen appears showing:
   - **Amount to Pay** (bill total in local and selected foreign currency)
   - **Enter Amount** field for entering the foreign currency value  

6. As you type the foreign amount:
   - The application shows the **Balance to be Paid** at the bottom  
   - If the full amount is entered → balance is `0`  
   - If partial → it shows the **remaining amount**

7. Click the **Proceed** button to complete the transaction.

---

This feature improves convenience for international customers by supporting seamless foreign currency billing.
