# With Bill Credit Note

This feature allows you to issue a **credit note** against a previously settled bill.

## 📝 Description

Used when a customer returns items and chooses to receive store credit instead of a cash refund.

---

## 🛠️ How to Use

1. Click the **Credit Note** button.
2. Select **With Bill Credit Note**.
3. Enter or scan the **Tax Invoice Number** of the bill.
4. Select items to include and set correct quantity.
5. Click **Create Credit Note**.
6. You'll be redirected to the billing screen.
7. Click **Save** on the bottom keyboard to complete.
